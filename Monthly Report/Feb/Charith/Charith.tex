\section{Project Overview}
This project focuses on biometric template generation using deep learning techniques with privacy-preserving cancelable biometrics. The system aims to develop a secure biometric authentication framework that protects user privacy while maintaining high recognition accuracy across multiple fingerprint datasets.

\section{Work Completed During Last 1 Month}
\begin{itemize}[leftmargin=*]

    \item Addressed reviewer comments for conference paper submission
    \item Trained models on both FVC2000 and FVC2006 fingerprint datasets
    \item Generated embeddings and performed initial evaluation metrics
\end{itemize}
